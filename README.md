# 🍽️ Real-Time Online Food Ordering System

## Overview

The **Real-Time Online Food Ordering System** is a web-based application developed using the **Django framework**. It allows users to register, log in, browse restaurants, view food menus, add items to the cart, complete the payment process, and place orders. All user and order information is stored in a SQLite database, making the application dynamic and database-driven.

---

## Features

* User Registration and Login
* Hotel Selection
* Hotel-Specific Food Menus
* Add to Cart Functionality
* Payment Page
* Order Confirmation
* SQLite Database Integration
* Responsive User Interface

---

## Technologies Used

* HTML
* CSS
* Bootstrap

---

## Project Workflow

1. User Registration/Login
2. Select a Hotel
3. Browse Food Menu
4. Add Items to Cart
5. Proceed to Payment
6. Place Order
7. Order Stored in Database
8. Order Confirmation Displayed

---

## Database

The project uses **SQLite** as the backend database to store:

* User Details
* Hotel Information
* Food Items
* Cart Details
* Order Details
